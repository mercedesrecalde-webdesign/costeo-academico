## 🚀 Cómo Usar PLATIFY

### Opción 1: Uso Local (Recomendado para ahora)

1. **Instalar servidor local simple:**
   ```bash
   npm install -g serve
   ```

2. **Ejecutar desde la carpeta `dist`:**
   ```bash
   cd "C:\Users\mmr_1\OneDrive\Escritorio\costeo recetas mrecalde\recipe-costing-app\dist"
   serve -s .
   ```

3. **Abrir en navegador:**
   - URL: `http://localhost:3000`
   - Listo! La app está corriendo

4. **Instalar como PWA** (Opcional):
   - Click en el ícono de instalar en la barra de direcciones
   - O: Menu Chrome (⋮) → "Instalar Platify"
   - Ahora tendrás un ícono en el escritorio

## 📤 Distribución para Venta

### Para Enviarlo a Clientes:

**Lo que necesitas entregar:**
1. Carpeta `dist` completa (ZIP)
2. Instrucciones de instalación (abajo)

### Archivo de Instrucciones para el Cliente:

```markdown
# PLATIFY - Instrucciones de Instalación

## ¿Qué necesitas?
- Navegador Chrome o Edge (actualizado)
- Node.js (solo para instalar el servidor)

## Instalación:

1. Descomprimir el archivo ZIP en tu escritorio

2. Instalar "serve" (solo una vez):
   - Abrir PowerShell
   - Ejecutar: `npm install -g serve`

3. Ejecutar PLATIFY:
   - Abrir PowerShell en la carpeta descomprimida
   - Ejecutar: `serve -s .`
   - Abrir navegador en: http://localhost:3000

4. OPCIONAL - Instalar como App:
   - En Chrome/Edge, click en "Instalar Platify"
   - Ahora aparecerá como app en tu inicio

## Soporte:
Email: tu_email@ejemplo.com
```
