# 🍽️ PLATIFY - Professional Recipe Costing App

**Aplicación profesional de costeo gastronómico para restaurantes, hoteles y operaciones F&B**

[![PWA Ready](https://img.shields.io/badge/PWA-Ready-success)](https://web.dev/progressive-web-apps/)
[![React](https://img.shields.io/badge/React-19.2-61DAFB)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-7.2-646CFF)](https://vitejs.dev/)

---

## ✨ Características Principales

### 📊 **Costeo Completo de Recetas**
- Cálculo automático de costo por porción
- Seguimiento de precios mayoristas
- Factores de corrección de ingredientes
- Soporte multi-moneda (USD, EUR, ARS, etc.)

### 🤖 **Generación de Recetas con IA**
- 8 tipos de dietas: Keto, Diabético, Sin Gluten, Sin Lactosa, Sin Huevos, Sin Frutos Secos, Vegetariano, Vegano
- 4 momentos del día: Desayuno, Almuerzo, Merienda, Cena
- 32+ plantillas pre-configuradas
- Cálculo automático de costos y nutrición

### 📱 **Progressive Web App**
- Instalable en cualquier dispositivo
- Funciona sin conexión
- Rápida y responsive
- Experiencia como app nativa

### 🌍 **Soporte Multiidioma**
- Español, English, Français, Português
- Fácil de agregar más idiomas

### 📈 **Análisis de Negocio**
- Reportes de costos
- Información nutricional
- Control de porciones
- Sugerencia de precios de venta

## 🚀 Inicio Rápido

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Preview de producción
npm run preview
```

## 🛠️ Stack Tecnológico

- React 19.2
- Vite 7.2
- vite-plugin-pwa + Workbox
- Lucide React (iconos)
- Recharts (gráficos)
- i18next (internacionalización)

## 📦 Estructura del Proyecto

```
src/
├── components/     # Componentes React
├── context/        # Context providers
├── data/          # Datos JSON
├── i18n/          # Traducciones
├── utils/         # Utilidades
└── App.jsx        # Componente principal
```

## 🌐 Deployment

### Netlify

3. Publish: `dist`
